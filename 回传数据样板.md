### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| shop_id | string | 店铺ID |
| good_id | string | 商品ID（与拉取时返回的 good_id 一致） |
| country | string | 国家代码 |
| trace_id | string | 追踪ID（与拉取时返回的 trace_id 一致） |
| down_stream_vendor_name | string | 固定值 `vendor_five` |
| content | string | tokege API 返回的完整 JSON（经 `JSON.stringify` 处理后的字符串） |
