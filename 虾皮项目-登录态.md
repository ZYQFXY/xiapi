# GoodDetailCallbackController

## GET 商品列表

GET /good/detail/user/list

### 请求参数

| 名称                      | 位置     | 类型      | 必选  | 说明                   |
| ----------------------- | ------ | ------- | --- | -------------------- |
| down_stream_vendor_name | query  | string  | 是   | 下游供应商名称，固定值（联系供应商提供） |
| Authorization           | header | string  | 是   | token，固定值：（联系供应商提供）  |
| pull_size               | query  | integer | 否   | 拉取数量，默认100，最大不超过100  |

> 返回示例

> 200 Response

```json
[
  {
    "shop_id": "",
    "good_id": "",
    "country": "",
    "trace_id": "",
    "add_time": ""
  }
]
```

### 返回结果

| 状态码 | 状态码含义                                                   | 说明   | 数据模型   |
| --- | ------------------------------------------------------- | ---- | ------ |
| 200 | [OK](https://tools.ietf.org/html/rfc7231#section-6.3.1) | none | Inline |

### 返回数据结构

状态码 **200**

| 名称          | 类型                                                        | 必选   | 约束   | 中文名 | 说明      |
| ----------- | --------------------------------------------------------- | ---- | ---- | --- | ------- |
| *anonymous* | [[ListGoodDetailListResp](#schemalistgooddetaillistresp)] | true | none |     | none    |
| » shop_id   | string                                                    | true | none |     | 店铺id    |
| » good_id   | string                                                    | true | none |     | 商品id    |
| » country   | string                                                    | true | none |     | 国家      |
| » trace_id  | string                                                    | true | none |     | traceId |
| add_time    | string                                                    | true | none |     | 添加时间    |

## POST 商品详情回调

> Body 请求参数

```json
{
  "shop_id": "string",
  "good_id": "string",
  "country": "string",
  "trace_id": "string",
  "down_stream_vendor_name": "string",
  "content": "string"
}
```

### 请求参数

| 名称            | 位置     | 类型                                                    | 必选  | 说明    |
| ------------- | ------ | ----------------------------------------------------- | --- | ----- |
| Authorization | header | string                                                | 是   | token |
|               |        |                                                       |     |       |
| body          | body   | [GoodCallbackResultReq](#schemagoodcallbackresultreq) | 否   | none  |

> 返回示例

> 200 Response

{}

## GET 统计供应商成功提交数量

GET /good/detail/user/stats/vendor-submission

### 请求参数

| 名称                      | 位置     | 类型      | 必选  | 说明        |
| ----------------------- | ------ | ------- | --- | --------- |
| down_stream_vendor_name | query  | string  | 是   | 供应商名称     |
| start_time              | query  | integer | 是   | 开始时间戳（毫秒） |
| end_time                | query  | integer | 是   | 结束时间戳（毫秒） |
| Authorization           | header | string  | 是   | none      |

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {
    "vendorName": "string",
    "takeCount": 0,
    "completeCount": 0,
    "checkSuccessCount": 0,
    "checkFailCount": 0,
    "completeRate": "string",
    "checkSuccessRate": "string",
    "startTime": "string",
    "endTime": "string"
  },
  "timestamp": 0
}
```

### 返回结果

| 状态码 | 状态码含义                                                   | 说明   | 数据模型 |
| --- | ------------------------------------------------------- | ---- | ---- |
| 200 | [OK](https://tools.ietf.org/html/rfc7231#section-6.3.1) | none |      |

## 属性

| 名称                | 类型             | 必选    | 约束   | 中文名 | 说明     |
| ----------------- | -------------- | ----- | ---- | --- | ------ |
| vendorName        | string         | false | none |     | 供应商名称  |
| takeCount         | integer(int64) | false | none |     | 领取数    |
| completeCount     | integer(int64) | false | none |     | 完成数    |
| checkSuccessCount | integer(int64) | false | none |     | 审核成功数  |
| checkFailCount    | integer(int64) | false | none |     | 审核失败数  |
| completeRate      | string         | false | none |     | 完成率    |
| checkSuccessRate  | string         | false | none |     | 审核成功率  |
| startTime         | string         | false | none |     | 查询开始时间 |
| endTime           | string         | false | none |     | 查询结束时间 |
